# 小红书文章模板

## 标题模板（20字以内）

### 问题型
- "为什么[现象]让[目标用户]困扰？"
- "[数字]个方法解决[痛点]"
- "揭秘[现象]背后的真相"

### 利益型
- "[时间/成本]搞定[目标]"
- "实测有效！[数字]个[好处]"
- "[工具/方法]让你的[结果]翻倍"

### 情感型
- "终于！[结果]让我[情感]"
- "[现象]太真实了"
- "[数字]年后，我终于[结果]"

### 数据型
- "[数字]%的人不知道的[技巧]"
- "只花[数字]元，达到[效果]"
- "[时间]内，[数字]人已[结果]"

---

## 正文结构模板

### 开头引入（50-100字）
**场景化开头：**
```
想象一下，[场景描述]。这不是科幻，而是[主题]的日常。
```

**痛点开头：**
```
你是否也有这样的困扰：[痛点描述]？最近发现[解决方案]，彻底改变了[结果]。
```

**数据开头：**
```
据[数据来源]，[数字]%的用户反映[现象]。今天分享[解决方案]。
```

### 正文主体（600-800字）

#### 🔥 现象/背景（100-150字）
```
🔥 [主题]最近有多火？

[时间]，[主题]迅速成为[领域]的现象级产品：
- 要点1：数据/事实
- 要点2：数据/事实
- 要点3：数据/事实
```

#### 💡 原因分析（150-200字）
```
💡 为什么[现象]如此重要？

✅ 原因1：[解释]
✅ 原因2：[解释]
✅ 原因3：[解释]
```

#### ⚡️ 核心内容/方法（200-300字）
```
⚡️ 如何实现[目标]？

方法1：[步骤/技巧]
- 具体操作1
- 具体操作2

方法2：[步骤/技巧]
- 具体操作1
- 具体操作2

方法3：[步骤/技巧]
- 具体操作1
- 具体操作2
```

#### 💰 价值/收益（100-150字）
```
💰 能带来什么实际价值？

• 时间节省：[具体数据/场景]
• 效率提升：[具体数据/场景]
• 创新潜力：[具体数据/场景]
```

#### ⚠️ 注意事项（100-150字）
```
⚠️ 使用时注意什么？

🔸 风险1：[风险] + [解决方案]
🔸 风险2：[风险] + [解决方案]
🔸 风险3：[风险] + [解决方案]
```

### 结尾互动（50-100字）

**提问式：**
```
[主题]是否会成为你的[好处]？[具体问题]？欢迎评论区交流！
```

**行动式：**
```
赶紧试试[方法]，分享你的[结果]！收藏本文，随时查看～
```

**悬念式：**
```
下期分享[预告]，关注我不错过！你觉得[主题]怎么样？评论区聊聊～
```

## 标签策略

### 3-5个标签组合

**组合1：宽泛+精准+热门**
- #AI工具（宽泛）
- #Clawdbot（精准）
- #效率神器（热门）

**组合2：领域+功能+人群**
- #科技（领域）
- #自动化（功能）
- #打工人必备（人群）

**组合3：话题+情感+行动**
- #AI助手（话题）
- #相见恨晚（情感）
- #值得收藏（行动）

### 标签研究方法
1. 使用小红书搜索框查看推荐标签
2. 参考同类高赞文章的标签
3. 平衡流量大的标签和精准标签
4. 避免不相关的热门标签（会被降权）

## 排版技巧

### Emoji使用指南

**列表符号：**
- 🔥 火爆/热门
- 💡 想法/原因
- ⚡️ 方法/技巧
- 💰 价值/收益
- ⚠️ 注意/风险
- ✅ 优点/正确做法
- ❌ 缺点/错误做法

**分隔符号：**
• • • • • • • • • • • • • •

**强调符号：**
👉 重点提示
📌 记住这点
⭐ 强调推荐
🎯 目标达成

### 段落格式
- 每段2-3句话
- 段落间空一行
- 重点内容加粗或用符号标记
- 使用数字列表（1. 2. 3.）
- 使用项目符号（• - ·）

## 发布检查清单

### 标题检查
- [ ] 20字以内
- [ ] 包含关键词
- [ ] 有吸引力/好奇心
- [ ] 表述清晰不模糊

### 内容检查
- [ ] 开头吸引人
- [ ] 结构清晰分段
- [ ] emoji使用合理
- [ ] 段落简短易读
- [ ] 有数据/案例支撑
- [ ] 结尾有互动

### 标签检查
- [ ] 3-5个标签
- [ ] 相关性强
- [ ] 宽泛+精准组合
- [ ] 包含热门标签

### 图片检查
- [ ] 图片清晰
- [ ] 风格统一
- [ ] 有文字说明
- [ ] 符合小红书规范
